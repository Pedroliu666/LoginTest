1.	Activate Virtual Environment:
  source .venv/bin/activate (macOS/Linux)
  .venv\Scripts\activate (Windows)

2.	Install Dependencies:
   pip install -r requirements.txt
	
3.	Build React Static Files:
   cd myfrontend / 
   npm install  / 
   npm run build

4.	Run Server:
   cd ..
   python manage.py / 
   runserver / 

6.	Run Tests:
   pytest
